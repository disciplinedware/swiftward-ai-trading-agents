## ERC-8004

- [8004.org](https://www.8004.org/) - Trustless Autonomous Agents on Open Protocols.
- [EIP-8004 Specification](https://eips.ethereum.org/EIPS/eip-8004) - Official EIP. Identity Registry, Validation Registry, Reputation Registry.
- [8004scan.io](https://www.8004scan.io/agents) - Agent explorer indexing all registries.
- [ERC-8004 Contracts (official)](https://github.com/erc-8004/erc-8004-contracts) - Contract source and ABIs. Deployed on Sepolia and 20+ chains.
- [ChaosChain Reference Implementation](https://github.com/ChaosChain/trustless-agents-erc-ri) - Production-ready reference, 74/74 tests passing.
- [Vistara 3-Agent Example](https://github.com/vistara-apps/erc-8004-example) - Alice/Bob/Charlie demo with market analysis + validation.
- [Awesome ERC-8004](https://github.com/sudeepb02/awesome-erc8004) - Curated list of SDKs, examples, tools.
- [Ethereum Magicians discussion](https://ethereum-magicians.org/t/erc-8004-trustless-agents/25098) - Official ERC discussion thread.
- [ERC-8004 Telegram](https://t.me/ERC8004) - Builder community.
